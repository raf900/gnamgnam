Collega GitHub per salvare

Per salvare le modifiche sul sito serve un token GitHub. Va fatto una volta sola: il token resta salvato su questo dispositivo.

  1. Apri github.com/settings/personal-access-tokens/new
  2. Repository access → "Only select repositories" → gnamgnam
  3. Permissions → Contents → Read and write
  4. Genera il token e incollalo qui sotto
